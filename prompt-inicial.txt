2️⃣ Según la familia profesional, cargar el experto:

* Informática → `agents/02-experto-informatica.md`

* **Turismo y Hostelería → `agents/07-experto-turismo.md`** ⭐ NUEVO

3️⃣ Según el curso:

* Segundo → `agents/04-segundo-curso.md`

4️⃣ Usar la plantilla:

`plantilla/index.html`

como base para generar el proyecto.

---

# 4. EL PROYECTO DEBE INCLUIR

* Contexto realista del proyecto
* Problema o necesidad empresarial
* Descripción completa del proyecto
* Tareas iniciales propuestas
* Orientaciones para el profesorado
* Sugerencias para que el **alumnado complete el desarrollo**
* Posibles entregables
* Metodología basada en **aprendizaje basado en proyectos**

Usar siempre:

* lenguaje inclusivo (profesorado / alumnado)
* contexto cercano al entorno empresarial

---

# 5. GUARDAR EL PROYECTO EN ESTA ESTRUCTURA

```
proyectos/
│
├── 01-Informatica/
├── 02-Administracion/
├── 03-Comercio/
├── 04-Turismo/
│
│   └── [CICLO]/
│       └── [CURSO]/
│           └── [CARPETA_PROYECTO]/
│               ├── proyecto.html
│               ├── logo.png
│               └── images/
```

proyectos/01-Informatica/DAM/Segundo/SEGUNDO-DAM-APP-GESTION/
