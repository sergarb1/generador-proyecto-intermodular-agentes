# ⚡ Inicio Rápido - Generador de Proyectos

## Para gente con prisa 👋

### ¿Qué necesitas?
- **Qwen Code** (gratis) - Se instala en 1 minuto

### ¿Qué haces?
1. Instalas Qwen
2. Ejecutas el script
3. Respondes 4 preguntas
4. ¡Proyecto listo!

---

## Paso 1: Instalar Qwen Code

Abre **PowerShell** (como administrador) y ejecuta:

```powershell
npm install -g @qwen-code/qwen-code
```

**¿Error?** Primero instala Node.js desde: https://nodejs.org/

---

## Paso 2: Ejecutar el Asistente

1. Abre esta carpeta en el explorador
2. Click derecho → **"Abrir en PowerShell"
3. Escribe:

```powershell
.\Iniciar-Asistente.ps1
```

---

## Paso 3: Responde las Preguntas

La IA te preguntará:

```
🤖 ¿Qué familia profesional necesita?
   1. Informática
   2. Administración

👤 Tú: 1

🤖 ¿Qué ciclo de Informática?
   - ASIR, SMR, DAM, DAW

👤 Tú: DAM

🤖 ¿Primer o Segundo curso?

👤 Tú: Segundo

🤖 ¿Alguna temática concreta? (opcional)

👤 Tú: [Enter para automático]
```

---

## Paso 4: ¡Listo!

```
✅ Proyecto generado en:
   proyectos-ejemplo/SEGUNDO-DAM-APP.html

📂 Abre el archivo para verlo
🖨️ Usa los botones PDF/Word para exportar
```

---

## ¿Problemas?

### "npm no se reconoce"
➡️ Instala Node.js primero: https://nodejs.org/

### "Qwen no se encuentra"
➡️ Ejecuta: `npm install -g @qwen-code/qwen-code`

### "Los acentos se ven raros"
➡️ Es normal, el HTML final se ve bien
