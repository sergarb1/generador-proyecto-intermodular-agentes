¿Qué es este proyecto?

Un sistema de generación de proyectos intermodulares asistido por IA para el IES Serra Perenxisa.

🎯

Objetivo

Facilitar la creación de proyectos intermodulares completos y pedagógicamente sólidos para el alumnado de Ciclos Formativos, sirviendo como guía y ejemplo.

🤖

Tecnología

Utiliza agentes de IA especializados (orquestador, experto en informática, experto en administración) para generar contenido adaptado a cada ciclo.

📁

Formato

Los proyectos se generan como HTML interactivo con navegación lateral, totalmente visualizables en navegador y exportables a Word/PDF.

✨

Enfoque

La introducción es detallada y realista, mientras que el resto de secciones son guías didácticas para que el alumnado complete la información.

⚙️

¿Cómo Funciona?

El proceso de generación sigue un flujo estructurado con agentes especializados.

1

Recopilación de Datos

El sistema pregunta al usuario:

  • • Familia profesional (Informática / Administración / Comercio)
  • • Ciclo formativo (ASIR, DAM, DAW, SMR, GM-ADM, GS-ASID, Comercio...)
  • • Curso (Primero / Segundo)
  • • Temática concreta (opcional)
2

Selección de Agentes

Según la familia, se activan los agentes especializados:

Informática

Experto Proyectos + Informática + Curso

Administración

Experto Proyectos + Administración + Curso

Comercio

Experto Proyectos + Comercio + Curso

3

Generación del Proyecto

Se crea la estructura completa:

  • • Introducción: Contexto empresarial/técnico detallado
  • • Resto de secciones: Guías didácticas para completar
  • • Archivos de datos: 12 archivos JavaScript con el contenido
4

Guardado y Visualización

El proyecto se guarda en:

proyectos/[FAMILIA]/[CICLO]/[CURSO]/[NOMBRE_PROYECTO]/

Incluye: proyecto.html, logo.png, carpeta data/, carpeta images/

🤖

Agentes de IA

Cada agente tiene una especialización concreta para generar contenido de calidad.

🎯

Orquestador

agents/00-orchestrator.md

Coordina todo el proceso, selecciona los agentes adecuados y valida que el proyecto cumpla los estándares de calidad.

💼

Experto Proyectos

agents/01-experto-proyectos.md

Diseña el contexto empresarial, la problemática y los objetivos del proyecto. Asegura que sea realista y pedagógico.

💻

Experto Informática

agents/02-experto-informatica.md

Define arquitecturas técnicas, tecnologías, implementación y mejores prácticas para proyectos IT.

📊

Experto Administración

agents/05-experto-administracion.md

Especializado en planes de empresa, análisis financiero, trámites legales y gestión empresarial.

🛒

Experto Comercio

agents/06-experto-comercio.md

Especialista en e-commerce, marketing digital, logística y atención al cliente online.

📚

Agentes por Curso

agents/03-primer-curso.md, 04-segundo-curso.md

Adaptan el contenido al nivel del curso (Primero o Segundo) ajustando la complejidad y profundidad.

🎨

Plantilla Premium

plantilla/index.html

Diseño visual profesional con Vue.js, Tailwind CSS, callouts, grids y tablas premium.
