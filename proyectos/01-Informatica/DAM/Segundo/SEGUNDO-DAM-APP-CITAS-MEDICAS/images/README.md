### Cómo descargar imágenes de Pexels para futuros proyectos:

1. Visita https://www.pexels.com/
2. Busca términos relevantes (ej: "medical app", "healthcare", "technology")
3. Haz clic en "Descargar gratis"
4. Guarda la imagen en la carpeta `images/` del proyecto
5. Actualiza `data/metadata.js` con la ruta local: `projectImage: "./images/hero.jpg"`
