## 🚀 Cómo Usar Este Proyecto

### Para el Alumnado

1. **Abrir el proyecto:**
   - Haz doble clic en `proyecto.html`
   - Se abrirá en tu navegador web
   - ¡No necesita internet para funcionar! (solo para las librerías CDN)

2. **Navegar por el contenido:**
   - Usa el menú lateral para moverte entre secciones
   - Puedes imprimir o exportar a Word con los botones superiores

3. **Personalizar:**
   - Edita `data/metadata.js` con tus datos:
     ```javascript
     studentName: "Tu Nombre",
     tutorName: "Nombre del Tutor/a",
     ```

4. **Completar las tareas:**
   - Cada sección tiene actividades marcadas con 📝
   - Completa las tablas vacías con información real
   - Sigue las checklist de cada sección

### Para el Profesorado

1. **Distribuir al alumnado:**
   - Copiar la carpeta completa a cada alumno/a
   - O subir a la plataforma virtual del centro

2. **Seguimiento:**
   - Revisar las tareas completadas en cada sección
   - Usar las rúbricas de evaluación incluidas

3. **Evaluar:**
   - Ver sección "Conclusiones" para rúbrica de evaluación
   - Revisar anexos completados
