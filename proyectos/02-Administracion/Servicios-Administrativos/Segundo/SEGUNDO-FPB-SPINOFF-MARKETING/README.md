## 🚀 Cómo Usar el Proyecto

### Para el Profesor/a

1. **Abrir el proyecto:**
   - Haz doble clic en `proyecto.html`
   - O ábrelo desde tu navegador (Chrome, Firefox, Edge)

2. **Navegar por el contenido:**
   - Usa el menú lateral para ir a cada sección
   - El proyecto tiene 9 secciones principales

3. **Imprimir o exportar:**
   - Botón "Imprimir" → Genera PDF o imprime en papel
   - Botón "Word" → Exporta a formato editable

## 🔧 Problemas Comunes

### El proyecto no se ve bien

- **Solución:** Asegúrate de abrir `proyecto.html` (no los archivos .js)
- **Requisito:** Necesitas conexión a internet la primera vez (para cargar Vue.js y Tailwind)

### Las imágenes no se ven

- **Solución:** Verifica que la carpeta `images/` esté junto a `proyecto.html`
- **Ruta:** Las imágenes deben estar en `./images/` relativo al proyecto

### Los datos no se cargan

- **Solución:** Todos los archivos `.js` deben estar en la carpeta `data/`
- **Orden:** El archivo `metadata.js` debe cargarse primero
