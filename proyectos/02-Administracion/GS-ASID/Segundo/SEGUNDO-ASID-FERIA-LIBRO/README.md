# Proyecto Feria del Libro - IES Serra Perenxisa

## 📁 Estructura del Proyecto

```
SEGUNDO-ASID-FERIA-LIBRO/
├── proyecto.html          # Archivo principal HTML
├── logo.png               # Logo del IES (copiar desde /plantilla/)
├── images/
│   └── hero.jpg           # Imagen principal (descargar de Pexels)
└── data/
    ├── metadata.js        # Metadatos del proyecto
    ├── resumen.js         # Contenido del resumen
    ├── introduccion.js    # Introducción y contexto
    ├── estado-arte.js     # Análisis del sector
    ├── viabilidad.js      # Estudio de viabilidad
    ├── requisitos.js      # Requisitos legales
    ├── diseno.js          # Diseño organizativo
    ├── implementacion.js  # Plan de implementación
    ├── administracion.js  # Gestión administrativa
    ├── herramientas.js    # Herramientas tecnológicas
    ├── conclusiones.js    # Conclusiones finales
    ├── bibliografia.js    # Referencias bibliográficas
    └── anexos.js          # Documentación complementaria
```

## 🚀 Cómo usar este proyecto

1. **Copiar el logo**: Copia el archivo `logo.png` desde la carpeta `/plantilla/` a esta carpeta.

2. **Descargar imagen hero**: Descarga una imagen de libro/lectura de [Pexels.com](https://www.pexels.com/es-es/buscar/fotos/libros/) y guárdala como `images/hero.jpg`

3. **Abrir el proyecto**: Abre el archivo `proyecto.html` en tu navegador.

4. **Personalizar**: Edita el archivo `data/metadata.js` con tus datos.
